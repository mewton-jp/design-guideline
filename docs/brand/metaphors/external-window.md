# Metaphor: External Window

## このメタファーが表す状態

External Window（外部への窓）は、  
このサイトの外側に存在する世界を  
**一時的に覗き見るための開口部**を表すメタファーである。

- 自分の場ではない
- 外の文脈を尊重する
- 中に引き込もうとしない
- 覗いたら、元の場所に戻ってこられる

外部リンクは遷移ではなく、  
**視線の移動**として扱う。

---

## 適用される範囲

- 外部サイトへのリンク
- 外部記事・リポジトリ・サービスへの参照
- SNS や他者の成果物への導線

External Window は常に、  
このサイトの **外部に向いた要素** として扱われる。

---

## OGP の位置づけ

外部リンクに付随する OGP は、  
「外の景色を切り取ったもの」として扱う。

- 自分で演出しない
- 加工・再構成しない
- 内容を要約しすぎない

OGP はプレビューではなく、  
**窓越しに見える風景**である。

---

## レイアウト

- 本文とは異なるリズムで配置されてよい
- Attached Photo と同様、少し目を引いてよい
- ただし、本文の主導線を奪ってはならない

External Window は、  
「ここから外が見える」と気づかせるために存在する。

---

## Voice & Tone

外部リンクに対して、  
評価や断言を添えない。

- 「おすすめ」「正解」と言い切らない
- 関心や文脈だけを簡潔に添える
- 判断は読み手に委ねる

リンクは紹介ではなく、  
**視線の誘導**に留める。

---

## Interaction

- 新しいタブ（別ウィンドウ）で開くかどうかは文脈によって決める
- 現在の文脈を破壊しない
- 戻ってくる前提を崩さない

External Window は出口ではない。  
**外を見たあと、戻ってこられる窓**である。

---

## 他のメタファーとの関係

- note  
  → 思考の途中で、外の事例を覗く窓
- sandbox  
  → 参考実装・事例を確認する窓
- mindmap  
  → 関連領域へ視野を広げる窓

External Window は、  
どの主メタファーにも **重ねて使えるサブメタファー** である。

---

## このメタファーでやらないこと

- 外部に引きずり出す構成を作らない
- リンク自体を主役にしない
- 過剰な説明や装飾を加えない
- 内部コンテンツと同一視しない

External Window は扉ではない。  
**覗くための窓**である。
